⚠️
Double generation bug
Never use langfuse.openai AND as_type="generation" on the same LLM call. Pick one. The drop-in wrapper is Method A, @observe with plain openai is Method B.
📍
propagate_attributes placement
Must wrap the call site of your top-level @observe function — not inside it. Tags and user_id won't appear in UI if placed incorrectly.
🔑
LANGFUSE_BASE_URL region
US → us.cloud.langfuse.com · EU → cloud.langfuse.com · JP → jp.cloud.langfuse.com. Wrong region = silent 401 errors.
🔢
BOOLEAN score value
Must be value=0 or value=1 (float) with data_type="BOOLEAN" explicitly set. Using Python's True/False causes silent type errors.
